How to use
----------------
Usage: FactorioApiConsoleApp [options]

Options:
  -a|--action <ACTION>  Values Allowed are 'check', 'start' or 'stop'
  -o|--otp <OTP>        6 Digit Otp
  -e|--email <EMAIL>    User's Email
  -?|-h|--help          Show help information


How to compile
----------------
 dotnet publish -c Release -r ubuntu.16.04-x64
 dotnet publish -c Release -r win10-x64
